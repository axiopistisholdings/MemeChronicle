# Code of Conduct

All participants in MemeChronicle are expected to:

- Be respectful and considerate in all communications.
- Avoid harassment, discrimination, and offensive language.
- Value constructive feedback and collaboration.
- Report unacceptable behavior via [memechronicle.com/about](https://memechronicle.com/about).

Violations may result in removal from the community and restriction of repository access.

Thank you for helping us maintain a welcoming and professional environment.
