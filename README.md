# MemeChronicle

Free automated meme coin news aggregator. Curated summaries from trusted publishers: Latest, Market Movers, Announcements & Top 10 Prices. No hype, just facts for traders.

## Public Marketing Website

This is the public-facing marketing website for [memechronicle.com](https://memechronicle.com), built with [Next.js](https://nextjs.org) and [Tailwind CSS](https://tailwindcss.com).

### Sections

- **Navbar** — Sticky nav with logo, anchor links, and "Launch App" CTA
- **Hero** — Tagline "Meme Coin News. No Hype." with primary CTA
- **Stats** — 100% Free · 24/7 Updates · 10+ Publishers · 0 Ads
- **Features** — Latest · Market Movers · Announcements · Top 10 Prices
- **How It Works** — Crawl → Summarise → Categorise
- **CTA** — Final call-to-action to the app
- **Footer** — Branding + link

### Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build

```bash
npm run build   # generates static export in /out
```

### Deploy

The site exports as fully static HTML (no server required). Deploy the `/out` folder to any CDN — [Vercel](https://vercel.com), Netlify, Cloudflare Pages, etc.
